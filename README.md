## MentorTech Landing Page

Landing Page de Captura de Leads para apresentar a proposta da MentorTech e converter visitantes em contatos qualificados.

### Objetivo do negocio

- Atrair potenciais clientes interessados em mentoria e solucoes tecnicas.
- Capturar dados de contato de leads por meio de formularios objetivos.
- Organizar o funil inicial para acompanhamento comercial.

### Tecnologias

- ASP.NET Core MVC
- Entity Framework Core
- MySQL
- Bootstrap 5

### Estrutura de pastas

- `Models`: entidades de dominio e modelos de dados.
- `Views`: paginas Razor da interface da landing page.
- `Controllers`: pontos de entrada HTTP e orquestracao das requisicoes.
- `Data`: contexto de banco, configuracoes e acesso a dados.
